# My-Team-Breakdown

## Description
This project was to create a team generator from the command line using node andjavascript. So a questioneer is generated asking hte party who they have as part of their team and relvent information that will be helpful with identifying that employee. Created test files to ensure code worked before putting into production steps. Many npm installations and jest were used as part of this application.
